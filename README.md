- 10. 差：阳气蠢辟于东，帝由群雍，物差其容。
- 11. 童：阳气始窥，物僮然咸未有知。
- 12. 增：阳气蕃息，物则增益，日宣而殖。

- 17. 徯：阳气有徯，可以进而进，物咸得其愿。

- 19. 进：阳引而进，物出溱溱，开明而前。

- 24. 争：阴气泛施，不偏不颇，物舆争讼，各遵其仪。
